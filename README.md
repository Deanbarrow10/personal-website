## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Basic knowledge of Astro/TypeScript (for modifications)

### Local Development
```bash
# Clone the repository
git clone https://github.com/Deanbarrow10/my_website.git
cd my_website

# Install dependencies
npm install

# Start development server
npm run dev
```

### Deployment
```bash
# Build for production
npm run build

# Deploy to Vercel
# Automatic deployment with Git integration
```

## 🔧 Customization

### Adding New Projects
1. Update the projects data in your Astro components
2. Add project SVG graphics to the assets folder
3. Update project information in the relevant TypeScript files

### Updating Content
- Personal information: Edit the about section components
- Contact information: Update contact section and links

### Styling Changes
- Colors: Modify Tailwind CSS configuration
- Typography: Update Tailwind font settings
- Layout: Adjust Tailwind utility classes for different arrangements

## 📊 Performance Optimizations

- **Static Site Generation**: Astro builds fast, static sites
- **SVG Graphics**: Vector graphics for crisp, lightweight visuals
- **Code Splitting**: Automatic optimization with Astro
- **Minimal JavaScript**: Only necessary interactive components
